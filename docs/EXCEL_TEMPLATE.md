# Excel Template

## Sheet 1: Company Details

Columns:

- Company Name
- GST Number
- Address
- Phone Number

## Sheet 2: Employee Details

Columns:

- Employee ID
- Employee Name
- Designation
- Department
- Email
- Mobile
- PAN
- UAN
- PF Number
- Bank Name
- Account Number
- IFSC
- Joining Date

- Employee ID

- HRA
- Conveyance Allowance
- Medical Allowance
- Special Allowance
- Bonus
- Incentive
- Overtime
- Other Earnings

## Sheet 4: Deductions

Columns:

- Employee ID
- PF
- ESI
- Professional Tax
- TDS
- Loan Recovery
- Advance Salary
- Other Deductions
